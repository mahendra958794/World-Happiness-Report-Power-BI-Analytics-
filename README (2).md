 Project Overview

This project presents an enterprise-level Power BI analytics report built on the World Happiness Dataset, covering 158 countries.
The goal is to understand global happiness patterns by analyzing economic, social, health, governance, and cultural factors through interactive dashboards and DAX-driven insights.


 Project Objectives

Compare happiness levels across countries and regions

Identify key drivers of happiness beyond GDP

Analyze inequalities between developed and developing regions

Study the role of trust, generosity, and social support

Evaluate data quality and unexplained (cultural) happiness


 Dataset Description

World Happiness Dataset includes:

Happiness Score & Rank

Economy (GDP per Capita)

Family (Social Support)

Health (Life Expectancy)

Freedom

Trust (Government Corruption)

Generosity

Dystopia Residual (Unexplained Happiness)

Standard Error (Survey Reliability)
